Microsoft Project

Microsoft Project je softver za upravljanje projektima, razvijen od strane Microsoft -a, koji je dizajniran da pomogne menadzerima projekta u razvoju plana, za dodeljivanje sredstava za zadatke, pracenje napretka, upravljanje budzetom, i analiziranje opterecenja.

Omogucava primenu savremenog koncepta upravljanja projektima. Intuitivan je, jednostavan za koriscenje i lici na programe iz Office paketa. Poseduje puno unapred definisanih izvestaja.

Microsoft Project ukljucuje nesmetan korisnicki interfejs poznat kao Ribbon Mogucnosti Interoperabiliti Microsoft Projekta su prosirena sa uvođenjem Microsoft Project Server-a i Microsoft Project Veb Access-a. Project Server smesta podatke u centralnu bazu podataka SQL, omogucavajuci nezavisne projekte za pristup zajednickim resursima. Web pristup omogucava ovlascenim korisnicima da pristupe bazu podataka Project Server-a preko Interneta i ukljucuje Timesheets, graficki analizu resursa opterecenja i administrativni tools.
